# Simple Product Manager

> Une petite application web pour ajouter des produits à une base de données MySQL via PHP e HTML.

---

## Description

Ce projet permet de gérer les produits dans une base de données MySQL.  
Fonctionnalité principale : **ajouter de nouveaux produits** avec nom, description et prix.

---

## Fonctionnalités

- Ajouter un produit avec :
  - Nom
  - Description
  - Prix
- Interface simple en HTML
- Stockage des produits dans MySQL via phpMyAdmin/XAMPP

---

## Technologies utilisées


- HTML 
- MySQL (via phpMyAdmin)
- Serveur local : XAMPP
